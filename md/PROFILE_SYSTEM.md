# Sistema de perfil profesional


Objetivo:

Convertir un CV humano en un perfil analizable.


Proceso:


CV PDF

↓

Extracción texto

↓

Análisis IA

↓

Perfil estructurado


Debe identificar:


## Información personal

- nombre
- ubicación
- contacto


## Experiencia

Por cada experiencia:

- empresa
- cargo
- duración
- responsabilidades
- logros


## Educación

- título
- institución
- cursos


## Skills


Clasificar:

Hard skills:

SQL
Python
Power BI
Excel


Soft skills:

comunicación
trabajo equipo


## Nivel estimado


Cada skill debe tener:

0-10


Ejemplo:


SQL:
8


Power BI:
8


Python:
6



Este perfil será utilizado para evaluar vacantes.
