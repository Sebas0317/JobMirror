# Job Monitor AI

## Objetivo

Construir una herramienta personal de monitoreo inteligente de ofertas laborales.

La herramienta NO busca reemplazar portales de empleo como LinkedIn, Computrabajo, Indeed, Magneto o Elempleo.

Su función es actuar como una capa inteligente encima de múltiples fuentes:

- recopilar vacantes
- organizar información
- analizar compatibilidad
- priorizar oportunidades


## Inspiración

El proyecto toma ideas de:

- Career-Ops
- AI Job Search

Especialmente:

- evaluación automática de ofertas
- pipeline de oportunidades
- scoring basado en perfil
- interfaz tipo comando/dashboard


## Filosofía

La herramienta debe responder:

"De todas las vacantes publicadas hoy, ¿cuáles realmente valen mi tiempo?"


## Alcance inicial

La primera versión debe permitir:

1. Cargar mi CV.
2. Extraer información profesional.
3. Crear un perfil estructurado.
4. Registrar vacantes.
5. Analizar compatibilidad.
6. Mostrar ranking de oportunidades.


## Fuera de alcance inicial

NO implementar todavía:

- envío automático de aplicaciones
- generación automática de CV
- cartas de presentación
- Telegram
- emails
- networking automático
- autofill de formularios


Estas funciones pueden agregarse posteriormente.


## Usuario objetivo

Inicialmente:

Un único usuario.

Uso personal.

El sistema debe estar preparado para soportar múltiples usuarios en el futuro, pero no implementarlo ahora.
