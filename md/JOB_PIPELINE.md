# Pipeline de vacantes


Flujo:


Nueva vacante

↓

Guardar

↓

Limpiar datos

↓

Extraer skills

↓

Comparar perfil

↓

Generar score

↓

Mostrar resultado



Cada vacante debe guardar:


id

titulo

empresa

ubicacion

fuente

fecha_publicacion

descripcion

url

skills_detectadas

score

estado



Estados:


NEW

ANALYZED

INTERESTING

IGNORED

APPLIED
